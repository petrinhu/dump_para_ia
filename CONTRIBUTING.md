# Como Contribuir

Obrigado pelo interesse em contribuir com dump_tree.sh!

## Reportar bugs

1. Abra uma issue
2. Descreva:
   - Versão do script
   - Comando usado
   - Saída/erro
   - Distro e versão do Bash
   - Passos para reproduzir

## Propor melhorias

1. Abra uma issue descrevendo a ideia
2. Se aceita, crie pull request com:
   - Código limpo e comentado
   - Testado em Fedora
   - Atualização no README se necessário

## Pull Requests

- Use branch separada (ex: feature/suporte-rust)
- Commits atômicos com mensagens claras
- Mantenha o estilo KISS: simples, direto, sem dependências novas
- Atualize CHANGELOG.md e teste com ./dump_tree.sh -g -n

Obrigado por contribuir!
